## Installation

How to run project locally:

1. Clone repository:

```bash
  git clone https://github.com/E1-XP/i-recruitment-task.git
```

2. Move inside project directory:

```bash
  cd i-recruitment-task
```

3. Install dependencies:

```bash
  npm install
```

4. Run development server:

```bash
  npm run dev
```
